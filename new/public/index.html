Componentes Compostos

CRUD Completo

Conjunto completo de operações CRUD (Create, Read, Update, Delete)

Estrutura:

  • routes/[nome].ts - Rotas para CRUD
  • controllers/[nome].ts - Controlador com métodos CRUD
  • services/[nome].ts - Serviço com operações de banco
  • models/[nome].ts - Modelo com validação Zod

Como usar:

// routes/user.ts
import { Router } from 'express';
import { UserController } from '../controllers/user';

const router = Router();
const controller = new UserController();

router.get('/users', controller.getAll);
router.post('/users', controller.create);
router.put('/users/:id', controller.update);
router.delete('/users/:id', controller.delete);

export const userRouter = router;

Busca Avançada

Sistema de busca com filtros, paginação e ordenação

Estrutura:

  • controllers/search.ts - Controlador de busca
  • services/search.ts - Serviço com lógica de busca

Como usar:

// Exemplo de uso na rota
router.get('/search', async (req, res) => {
  const { query, page, limit, sort } = req.query;
  const results = await searchService.search({
    query,
    page: Number(page),
    limit: Number(limit),
    sort
  });
  res.json(results);
});

Autenticação

Sistema completo de autenticação com JWT

Estrutura:

  • middlewares/auth.ts - Middleware de autenticação
  • services/auth.ts - Serviço de autenticação
  • models/user.ts - Modelo de usuário

Como usar:

// Proteger uma rota
router.get('/protected',
  authMiddleware,
  async (req, res) => {
    // Rota protegida
  }
);

Componentes Básicos

Rota

Define endpoints da API com Express Router

Como usar:

// routes/example.ts
import { Router } from 'express';
const router = Router();

router.get('/example', handler);
export const exampleRouter = router;

Controlador

Gerencia a lógica de requisições e respostas

Como usar:

// controllers/example.ts
export class ExampleController {
  async getAll(req: Request, res: Response) {
    const items = await service.getAll();
    res.json(items);
  }
}

Serviço

Implementa a lógica de negócio

Como usar:

// services/example.ts
export class ExampleService {
  async getAll() {
    return await db.find();
  }
}
